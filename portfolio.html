Gimme DOM (WIP)

A Vaadin 8 extension enabling developers to have more control over the web DOM. My goal here is to solve Vaadin 8's core problem: it does not feel like you are developing web applications. For example, some components lack common event handlers. Others lack important DOM event handlers altogether. Vaadin does not enable developers to modify DOM directly. Gimme DOM aims to implement this.

Java String

A set of string-central classes and methods. Features include, but not limited to: case modifier (e.g. snake_case to camelCase), string tokenizer, and miscellaneous string methods such as isAnagram, isPalindrome, and an indexOf method supporting Regular Expression.
