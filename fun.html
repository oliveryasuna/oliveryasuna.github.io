Counting

Here are some fun counting problems.

  1. How many \(3\)-digit even numbers are there?

    2. There are three spaces (for \(3\)-digits), when read left-to-right, represent the number's place value \(\left\{\text{hundreds},\text{tens},\text{ones}\right\}\).

    The question can be answered by working right-to-left:

    The third space's digit must be even for the number to be even. Therefore, the third space's digit must be in the set \(\left\{0,2,4,6,8\right\}\) (\(5\) choices).

    The second space's digit can be any single-digit. Therefore, the second space's digit must be in the interval \(\left[0,9\right]\) (\(10\) choices).

    The first space's digit must be positive. Therefore, the first space's digit must be in the interval \(\left[1,9]\right]\) (\(9\) choices).

    Now, the number of choices for each space are multipled:

    \(5*10*9=450\)

    Therefore, the answer is \(450\) numbers.
